ZipZap allows users to find skateboarding accessory brands at local retailers near their current location.

ZipZap makes use of XML data feeds providing always-up-to-date retailer information and special promotions, interfaces with the Location API in iOS, and features several unique interface elements.

This app contained unique searching user interfaces that helped filter the brand selections.
